To create a more effective heuristics function, we can incorporate some common heuristics used in graph theory, such as the concept of the "Total Weighted Shortest Path" (TWSP) or "All-Pairs Shortest Paths" (APSP). This approach not only considers the direct distance between nodes but also the indirect paths that might lead to a higher total cost.

Here's a Python function named `heuristics()` that employs the "Total Weighted Shortest Path" concept. This function calculates a heuristic score for each edge by summing up the total cost from the source node to all other nodes via that edge.

```python
import numpy as np

def all_pairs_shortest_paths(distance_matrix):
    n = len(distance_matrix)
    dist = distance_matrix.copy()
    for k in range(n):
        for i in range(n):
            for j in range(n):
                dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
    return dist

def heuristics(distance_matrix):
    # Calculate the all-pairs shortest paths
    ap_shortest_paths = all_pairs_shortest_paths(distance_matrix)
    
    # Calculate the heuristic score for each edge by summing the total weighted paths
    heuristics_matrix = np.zeros_like(distance_matrix)
    for i in range(len(distance_matrix)):
        for j in range(len(distance_matrix[i])):
            if i == j:
                heuristics_matrix[i][j] = 0
            else:
                heuristics_matrix[i][j] = distance_matrix[i][j] + ap_shortest_paths[i][j]
                
    return heuristics_matrix

# Example usage
distance_matrix = np.array([[0, 10, 20, 30],
                           [10, 0, 5, 15],
                           [20, 5, 0, 25],
                           [30, 15, 25, 0]])

print(heuristics(distance_matrix))
```
