```python
import numpy as np

def manhattan_heuristics(distance_matrix):
    # Checking for valid input
    assert distance_matrix.ndim == 2, "Input matrix should be 2D"
    n_vertices = distance_matrix.shape[0]
    
    # Calculate Manhattan distance heuristic
    manhattan_dist = np.sum(np.abs(np.arange(n_vertices)[:, np.newaxis] - np.arange(n_vertices)[np.newaxis, :]), axis=2)
    
    # Scale the heuristic to ensure it is comparable in magnitude to the distance matrix
    max_dist = np.max(distance_matrix)
    normalized_heuristics = manhattan_dist / max_dist
    
    return normalized_heuristics

# Test the function
distance_matrix = np.array([[0, 10, 15, 20],
                             [10, 0, 35, 25],
                             [15, 35, 0, 30],
                             [20, 25, 30, 0]])

print(manhattan_heuristics(distance_matrix))
```

This new heuristics() function uses the Manhattan Distance heuristic, which is known to outperform simple summed Euclidean distances for TSP. By normalizing the result with the maximum distance in the input matrix, we ensure that the heuristic values are comparable to the actual distances and are directly usable in an A* search algorithm.
