```python
def heuristics(distance_matrix):
    # Calculate Manhattan Distance
    manhattan_distances = np.sum(np.abs(distance_matrix), axis=0)
    
    # Max possible distance in a matrix (sum of all distances)
    max_distance = np.sum(distance_matrix)
    
    # Undesirability score for each edge (row)
    undesirability_scores = max_distance - manhattan_distances
    
    # Ensure the score is always non-negative
    undesirability_scores = np.maximum(undesirability_scores, 0)
    
    return undesirability_scores
```
