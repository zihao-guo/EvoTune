```python
def heuristics(distance_matrix):
    # Calculate the sum of each row
    row_sums = np.sum(distance_matrix, axis=1)
    
    # Create a new matrix where each entry corresponds to the undesirability score
    # of including the corresponding edge in the solution
    undesirability_scores = 1 / row_sums
    
    # Ensure the matrix is non-negative by setting any potential negative scores to 0
    undesirability_scores[undesirability_scores < 0] = 0
    
    return undesirability_scores
```

This function creates a new distance matrix where each entry represents the reciprocal of the row sum in the original distance matrix. The reciprocal values indicate how much that row contributes to the total distance of its vertices. Therefore, edges with higher row sums (higher distance to their vertices) will have smaller undesirability scores, meaning they are less undesirable to include in a solution.
