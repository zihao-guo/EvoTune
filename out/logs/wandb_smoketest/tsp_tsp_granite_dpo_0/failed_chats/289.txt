```python
def heuristics(distance_matrix):
    import numpy as np

    # Ensure the matrix is square
    n = distance_matrix.shape[0]
    if n!= distance_matrix.shape[1]:
        raise ValueError("Input matrix should be square.")

    # Calculate the heuristic score
    heuristic_scores = np.zeros((n, n))

    for i in range(n):
        for j in range(n):
            if i!= j:
                # Neighbors count
                neighbor_count = np.sum(distance_matrix[i] < distance_matrix[j])
                
                # Weight based on inverse distance to neighbors
                weight = 1 / (distance_matrix[j] + 1)

                # Combine with absolute distance
                heuristic_scores[i, j] = distance_matrix[i, j] + neighbor_count * weight

    return heuristic_scores
```

3. We add the neighbor count and weighted distance to get the heuristic score, thus combining both local (relative position) and global (absolute distance) information about each edge.
