The Edge Replacement Property states that in any optimal solution to a combinatorial optimization problem (like minimum spanning tree), no edge can be removed and replaced by another edge that has a smaller weight, without violating optimality.

Here's a Python function that implements the aforementioned approach. This function calculates a heuristic score for each edge by considering its weight and its degree in the graph (weighted degree centrality).

```python
import numpy as np

def heuristics(distance_matrix):
    # Check if the input is a 2D numpy array
    assert len(distance_matrix.shape) == 2, "Input should be a 2D array"
    
    n = distance_matrix.shape[0]
    # Create adjacency matrix from distance matrix
    adjacency_matrix = np.zeros((n, n))
    for i in range(n):
        for j in range(i+1, n):
            if distance_matrix[i][j]!= 0:
                adjacency_matrix[i][j] = distance_matrix[i][j]
                adjacency_matrix[j][i] = distance_matrix[i][j]  # Assuming undirected graph
    
    # Calculate weighted degree centrality for each node
    weighted_degree_centrality = np.sum(adjacency_matrix, axis=1)
    
    # Calculate heuristic score for each edge
    heuristic_scores = np.zeros_like(distance_matrix)
    for i in range(n):
        for j in range(i+1, n):
            # Score is the product of the edge weight and the sum of weights of its neighbors
            heuristic_scores[i][j] = distance_matrix[i][j] * weighted_degree_centrality[i] + 
                                 distance_matrix[j][i] * weighted_degree_centrality[j]
            heuristic_scores[j][i] = heuristic_scores[i][j]  # Assuming undirected graph
    
    return heuristic_scores
```

This function considers both the edge weight and its impact in the global structure (sum of weights of its neighbors), potentially providing a better heuristic than the given solutions.
