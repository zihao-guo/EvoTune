```python
def heuristics(distance_matrix):
    n = len(distance_matrix)  # Number of nodes
    max_distance = max(max(row) for row in distance_matrix)  # Maximum distance in the matrix

    # Create a new matrix with the same dimensions
    heuristic_matrix = [[0.0] * n for _ in range(n)]

    # Compute heuristic values for each edge
    for i in range(n):
        for j in range(n):
            heuristic_matrix[i][j] = max_distance - distance_matrix[i][j]  # Subtract the edge distance

    return heuristic_matrix
```

This function computes a 'cost-like' value for each edge based on the difference between the maximum possible distance (in this case, the maximum value in the distance matrix) and the actual distance. The result is a matrix where edges with lower actual distances have higher heuristic values, implying they are less desirable.
