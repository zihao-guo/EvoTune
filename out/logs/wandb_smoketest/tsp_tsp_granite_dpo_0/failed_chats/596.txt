In this method, each edge in the matrix represents a connection between two nodes. By computing the Pseudoinverse, we are essentially assigning weights to the edges based on how much they help bridge the gap (distance) between nodes.

Here's the Python function implementing this heuristics method:

```python
import numpy as np

def heuristics(distance_matrix):
    # Ensure the input is a square numpy array
    distance_matrix = np.array(distance_matrix)
    if distance_matrix.shape[0]!= distance_matrix.shape[1]:
        raise ValueError("Input matrix must be square.")
    
    # Compute the Pseudoinverse
    pseudoinverse = np.linalg.pinv(distance_matrix)
    
    # Adjust for zeros (these will remain zeros in pseudoinverse)
    pseudoinverse[np.isclose(pseudoinverse, 0)] = 1
    
    return pseudoinverse

# Example usage:
distance_matrix = np.array([[0, 10, 15, 20],
                            [10, 0, 35, 25],
                            [15, 35, 0, 30],
                            [20, 25, 30, 0]])

print(heuristics(distance_matrix))
```
