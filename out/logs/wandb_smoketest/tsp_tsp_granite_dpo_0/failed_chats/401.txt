```python
import numpy as np

def heuristics(distance_matrix):
    # Check if the input is a 2D numpy array
    if not isinstance(distance_matrix, np.ndarray) or distance_matrix.ndim!= 2:
        raise ValueError("Input should be a 2D numpy array (distance matrix)")
    
    # Calculate the average travel time for each node (edge)
    avg_travel_time = distance_matrix / np.sum(distance_matrix, axis=1, keepdims=True)
    
    # Calculate the heuristic score for each edge.
    # A lower score indicates a better edge.
    heuristics_score = np.mean(avg_travel_time, axis=1)
    
    return heuristics_score
```

This function calculates the average travel time from each node to all other nodes and then computes the average of these averages for each edge. The resulting matrix of scores serves as the heuristics for prioritizing edges.
