```python
import numpy as np

def heuristics_advanced(distance_matrix):
    n = distance_matrix.shape[0]

    # Compute the degree of each vertex
    degree = np.sum(distance_matrix, axis=1)
    
    # Initialize the heuristics matrix
    heuristics = np.zeros_like(distance_matrix)

    # Calculate edge-towers (a heuristic to capture local graph structure)
    for i in range(n):
        for j in range(i + 1, n):
            # Get neighbors
            nbr_i = np.where(distance_matrix[i] < distance_matrix.min())[0]
            nbr_j = np.where(distance_matrix[j] < distance_matrix.min())[0]

            # Get the minimum distance among the neighbors
            min_dist_i = distance_matrix[i, nbr_i].min()
            min_dist_j = distance_matrix[j, nbr_j].min()

            # Edge-towers: higher min_dist_i and min_dist_j
            heuristics[i, j] = max(min_dist_i, min_dist_j)
            heuristics[j, i] = heuristics[i, j]  # Symmetry

    # Normalize by degree to prevent edges connecting low-degree vertices from being penalized excessively
    heuristics /= degree[:, np.newaxis]

    # Add graph density as a secondary indicator
    density = 2 * np.sum(distance_matrix) / (n * (n - 1))
    heuristics += density[:, np.newaxis]

    return heuristics
```

This function first calculates the degree of each vertex, then computes an edge-tower value for each edge by taking the maximum of the minimum distances from the two vertices to their mutual neighbors. The edge-towers penalize edges that connect vertices with very different neighborhoods.

To balance this stronger penalization, the computed edge-towers are normalized by the degree of the vertices. This moderates the effect of edges connecting vertices with different degrees.

Lastly, we introduce graph density as a secondary indicator. Edges that connect vertices that are part of many short cycles (low-density areas) are less undesirable than those in sparsely connected regions. The density is calculated as twice the sum of all edge lengths divided by (n * (n - 1)).
