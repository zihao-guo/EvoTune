```python
import numpy as np

def heuristics(distance_matrix):
    n = len(distance_matrix)
    # Initialize a matrix to store the f-scores (heuristics)
    f_scores = np.zeros_like(distance_matrix)

    # Apply Nearest Neighbor (NN) move
    for i in range(n):
        f_scores[i] = distance_matrix[i][i]

    # Apply 2-opt move to iteratively improve the solution
    for _ in range(1000):  # Arbitrarily set the number of iterations
        improved = False
        for i in range(n):
            for j in range(i+1, n):
                # Swap the i-th and j-th elements
                temp = distance_matrix[i][j]
                distance_matrix[i][j] = distance_matrix[j][i]
                distance_matrix[j][i] = temp

                # Calculate the new f-score
                new_f_score = 0
                for k in range(n):
                    new_f_score += min(distance_matrix[i][k], distance_matrix[j][k])

                # If the new f-score is lower than the old one, update the matrix
                if new_f_score < f_scores[i] + f_scores[j]:
                    f_scores[i] = new_f_score
                    f_scores[j] = new_f_score
                    improved = True

        # If no improvement, the current solution is the best we can do
        if not improved:
            break

    return f_scores
```

This function initializes each node with its own distance, then iteratively applies the 2-opt move (swapping pairs of nodes in the distance matrix) to potentially reduce the total weight. The process is repeated for a fixed number of iterations (1000 in this case), and the best solution is kept.
