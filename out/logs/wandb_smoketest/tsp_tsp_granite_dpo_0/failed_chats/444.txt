```python
def heuristics(distance_matrix):
    import numpy as np

    # Compute the degree of each node (number of edges connected)
    node_degrees = np.sum(distance_matrix, axis=0)

    # Calculate the weighted score for each edge
    heuristic_matrix = np.zeros_like(distance_matrix)
    for i in range(distance_matrix.shape[0]):
        for j in range(distance_matrix.shape[1]):
            # Avoid adding edges to the same node (cycle)
            if i == j:
                heuristic_matrix[i, j] = 1e10
            else:
                # Higher degree nodes might be more critical
                edge_weight = 1 / node_degrees[i] if i!= j else 1e10  # Symmetry for edges to the same node
                heuristic_matrix[i, j] = 1 / distance_matrix[i, j] * edge_weight

    return heuristic_matrix
```

This function assigns higher scores to edges that connect high-degree nodes, which generally indicate a more central role in the graph. This approach can help identify potential bottlenecks and improve the quality of the solution. The score for an edge is calculated as the reciprocal of the edge's weight (distance) multiplied by a factor based on the node's degree.
