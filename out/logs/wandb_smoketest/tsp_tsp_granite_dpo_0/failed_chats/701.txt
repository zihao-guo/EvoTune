```python
import numpy as np

def heuristics(distance_matrix):
    # Calculate the sum of each row (importance score)
    row_sums = np.sum(distance_matrix, axis=1)
    # Subtract the minimum distance in each row to normalize
    distance_matrix -= np.min(distance_matrix, axis=1, keepdims=True)
    # Calculate the importance score: lower is higher importance
    importance_score = row_sums / np.sum(row_sums)
    # Return the result as the heuristic matrix
    return importance_score

# Test the function
distance_matrix = np.array([[0, 10, 15, 20],
                            [10, 0, 35, 25],
                            [15, 35, 0, 30],
                            [20, 25, 30, 0]])
print(heuristics(distance_matrix))
```

This function works by first calculating the sum of each row (which represents the total importance of all edges connected to that node). Then, it subtracts the minimum distance in each row to normalize the values, assuming edges with minimum distances are already 'important' based on the network structure. Finally, it divides by the sum of each row to get a final importance score. Lower scores indicate higher importance, i.e., edges with lower distances are considered more important.
