def heuristics(distance_matrix):
    # Compute all-pairs shortest paths (Floyd-Warshall algorithm)
    dist = distance_matrix

    dist = np.minimum(dist, np.linalg.norm(dist, axis=1))
    
    # Calculate centrality scores for each node
    centrality = np.zeros((n, n))
    for i in range(n):

            centrality[i][j] = dist[i][j]**2 / np.sum(dist**2)

    # Create a heuristics matrix where each entry represents the undesirability

    heuristics_matrix = np.zeros_like(distance_matrix)
    for i in range(n):

            heuristics_matrix[i][j] = centrality[i][j]

    return heuristics_matrix

This function first computes the shortest paths between all pairs of nodes using the Floyd-Warshall algorithm. Then, it calculates a centrality score for each node by normalizing the square of the shortest-path distance to all other nodes. Edges in the graph connect nodes with high centrality scores, and these edges are less undesirable to include.

The higher the centrality score, the lower the value in the heuristics matrix, indicating that including this edge is less undesirable.
