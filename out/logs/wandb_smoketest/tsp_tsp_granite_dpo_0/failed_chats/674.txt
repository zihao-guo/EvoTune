Here is the proposed `heuristics` function:

```python
import numpy as np

def heuristics(distance_matrix):
    # Step 1: Compute the average distance to neighboring nodes
    avg_neighbor_distance = np.mean(distance_matrix, axis=1)
    
    # Step 2: Compute the density of neighboring nodes for each node
    node_densities = np.sum(distance_matrix > 0, axis=1) / (distance_matrix.shape[1] - 1)
    
    # Step 3: Combine these two factors into a heuristic score
    heuristic_scores = -avg_neighbor_distance + node_densities  # Lower scores are better
    
    # Ensure the heuristic scores are non-negative
    heuristic_scores = np.clip(heuristic_scores, 0, np.inf)
    
    return heuristic_scores
```

This function works as follows:

1. `avg_neighbor_distance`: For each node, it computes the average distance to its neighboring nodes. This metric is designed to encourage connections that are relatively close to the node.

2. `node_densities`: It calculates the density of neighboring nodes for each node. The density is the ratio of the number of connected nodes (excluding the node itself) to the total number of possible neighboring nodes (minus one, to exclude the node from itself).

3. `heuristic_scores`: It creates the final heuristic scores by adding the negative of `avg_neighbor_distance` and `node_densities`. The negative sign ensures that lower scores (i.e., smaller, better heuristic values) are favored.
