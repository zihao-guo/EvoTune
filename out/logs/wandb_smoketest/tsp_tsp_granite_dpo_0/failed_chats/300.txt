```python
def weighted_distance_heuristics(distance_matrix, weights=1):
    """
    This function calculates a heuristic score for each edge in a distance matrix.
    Edges with smaller distances get higher weights, encouraging their inclusion.
    
    Args:
    distance_matrix (list of list): A 2D list representing the distance between each pair of nodes.
    weights (float): The weight factor to apply to the distance (default is 1).
                      Higher weights give more importance to shorter edges.
    
    Returns:
    list of list: A 2D list with the same shape as the input, where each element is the weighted distance.
    """
    heuristic_matrix = [[distance * weights for distance in distance_row] for distance_row in distance_matrix]
    return heuristic_matrix
```
