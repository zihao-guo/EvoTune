The Potential for Reduction (PFR) measures the difference between the distance to the nearest existing node and the distance to the proposed new node. A lower PFR indicates a more promising candidate for reducing the total cost.

The Connectedness Index (CI) measures how well a node connects to the existing graph. It helps identify the most connected nodes that can potentially form the backbone of the solution.

Here's the Python function incorporating these heuristics:

```python
import numpy as np

def PFR(distance_matrix, new_node):
    # Calculate the PFR for each existing node
    distances_to_existing = np.min(distance_matrix, axis=0)
    pfr_values = distance_matrix[new_node] - distances_to_existing
    
    return pfr_values

def CI(distance_matrix, new_node):
    # Calculate the Connectedness Index
    num_connections = np.sum(distance_matrix[new_node] < np.Inf)
    total_connections = np.sum(distance_matrix!= np.Inf)
    return num_connections / total_connections if total_connections > 0 else 0

def heuristics(distance_matrix):

    new_node_index = num_nodes - 1

    pfr_values = PFR(distance_matrix, new_node_index)

    ci_values = CI(distance_matrix, new_node_index)
    
    # Combine PFR and CI as a single heuristic value
    heuristic_values = (1 - pfr_values) * ci_values  # Normalize with CI values
```
