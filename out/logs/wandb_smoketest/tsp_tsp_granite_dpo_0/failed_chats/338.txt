Edge betweenness centrality measures the number of shortest paths that pass through a given edge in a graph. Edges with higher betweenness centrality are considered more critical in terms of their influence on overall connectivity and, in our context, may be more undesirable to include in a solution.

```python
import numpy as np
from scipy.sparse.csgraph import csgraph_from_dense

def heuristics(distance_matrix):
    # Convert the distance matrix to a sparse graph adjacency matrix
    adjacency_matrix = csgraph_from_dense(distance_matrix > 0)
    
    # Calculate the edge betweenness centrality using the Brandes algorithm
    edge_betweenness, _ = csgraph_betweenness(adjacency_matrix)
    
    # Convert the edge betweenness centrality to a matrix of the same shape as the input
    max_betweenness = np.max(edge_betweenness)
    heuristics_matrix = np.zeros_like(distance_matrix)
    
    for i in range(distance_matrix.shape[0]):
        for j in range(distance_matrix.shape[1]):
            if i!= j:  # Ignore diagonal entries
                heuristics_matrix[i, j] = edge_betweenness[i, j] / max_betweenness
    
    return heuristics_matrix
```
