Here is a novel heuristics function that considers the average distance in the matrix and scales it to a range between 0 and 1, representing an indicator of how undesirable each edge is:

```python
def heuristics(distance_matrix):
    n = len(distance_matrix)
    avg_distance = sum(sum(distance_matrix)) / (n * (n - 1))
    heuristics_matrix = [(avg_distance - d) / avg_distance for d in distance_matrix]
    return heuristics_matrix
```

In this function, the average distance `avg_distance` is calculated first. Then, for each edge (represented by a distance `d` in the matrix), it subtracts the average distance from `d` and scales the result by dividing it with the average distance. This scaling ensures that the heuristic values fall in the range of 0 to 1. A value closer to 1 indicates that the corresponding edge is more desirable to include (as it is less undesirable compared to the average distance), while a value closer to 0 signifies that the edge is more undesirable.
