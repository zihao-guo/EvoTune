```python
def heuristics(distance_matrix):
    n = len(distance_matrix)
    # Initial value to a large number
    max_score = 100000
    
    # Calculate weighted sum of distances
    for i in range(n):
        for j in range(i+1, n):
            weight = (1 / (distance_matrix[i][j] + 1))  # Using a small value to avoid division by zero
            max_score = min(max_score, distance_matrix[i][j] * weight)
    
    # Add a simple connectivity factor
    connectivity_score = 0
    for k in range(n):
        for l in range(k+1, n):
            connectivity_score += min(distance_matrix[k][l], distance_matrix[l][k])
    
    max_score += connectivity_score
    
    # Normalize the score to be between 0 and 1
    max_score /= (n * (n-1) / 2)
    
    return max_score
```

In this function, we start with a large initial score (100000) and iteratively update it. For each edge `(i, j)`, we calculate a weight as `(1 / (distance_matrix[i][j] + 1))`, giving more importance to edges with smaller distances. We then update the maximum score by taking the minimum of the current maximum and `distance_matrix[i][j] * weight`.

Additionally, we compute a connectivity score by summing up the minimum of distances between every pair of nodes. This encourages the function to favor connections that improve overall network connectivity.

Finally, we normalize the score by dividing it by `(n * (n-1) / 2)` to ensure the result is within the range [0, 1], where 0 indicates an ideal solution and 1 represents the worst possible solution.
