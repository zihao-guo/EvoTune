To create a more effective heuristics function, we could employ the concept of the Minimum Spanning Tree (MST) and the concept of edge weights. The idea is to assign higher negative scores to less desirable edges based on their weights and the overall structure of the MST.

Here is a Python function using the Kruskal's algorithm to find the Minimum Spanning Tree:

```python
import heapq

def heuristics(distance_matrix):
    n = len(distance_matrix)
    # Convert distance matrix into a list of edges and their weights
    edges = [(distance_matrix[i][j], i, j) for i in range(n) for j in range(i+1, n)]
    # Sort edges by weight
    edges.sort()

    # Initialize MST and result matrix
    mst = []
    heuristics_matrix = [[0]*n for _ in range(n)]
    
    # Kruskal's algorithm
    for edge_weight, u, v in edges:
        if len(mst) < n-1:
            # Add the edge if it doesn't form a cycle
            if not any(mst[i][1] == u and mst[i][2] == v for i in range(len(mst))):
                mst.append(edge_weight)
                heapq.heappush(edges, (0, u, v))  # Re-add edge with 0 weight for next iterations
        else:
            # If MST is complete, stop
            break

    # Assign heuristics
    for u, v in mst:
        heuristics_matrix[u][v] = distance_matrix[u][v] - heuristics_matrix[v][u]  # Make sure heuristics are negative for better scoring

    # For edges not in MST, assign higher negative values
    for u, v in [(u, v) for u in range(n) for v in range(u+1, n) if (u, v) not in edges]:
        heuristics_matrix[u][v] = -distance_matrix[u][v]

    return heuristics_matrix
```

This function first uses Kruskal's algorithm to build a Minimum Spanning Tree from the input distance matrix. It then assigns heuristic values to each edge. Edges in the MST are given a heuristic value equal to the negative of their weight (which makes them desirable) and edges not in the MST are given higher negative heuristic values (undesirable). This approach tries to balance between preserving the integrity of the MST (which represents the most efficient set of connections) and providing a clear notion of undesirable edges, making it more robust and performant than simple distance matrix transformations.
