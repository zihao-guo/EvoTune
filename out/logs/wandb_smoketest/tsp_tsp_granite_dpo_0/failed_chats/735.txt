This method calculates a score for each edge based on the potential saving if that edge were included in the minimum spanning tree of the graph.

Here's how I'll implement it:

1. I'll first compute the total weight of an edge (its distance) from the given distance matrix.
2. Then, I'll calculate the "cheapest" edge to add to form a spanning tree, which is the minimum total weight edge that doesn't form a cycle.
3. The heuristic score for each edge will be the difference between the total weight of the current edge and the cheapest edge to add.

```python
import numpy as np

def cheapest_insertion(distance_matrix):
    # Get the number of nodes (vertices) in the graph
    n = distance_matrix.shape[0]
    
    # Initialize variables
    parent = np.zeros(n, dtype=int)
    rank = np.zeros(n, dtype=int)
    min_edge_weight = float('inf')
    
    def find(x):
        if parent[x]!= x:
            parent[x] = find(parent[x])
        return parent[x]

    def union(x, y):
        root_x = find(x)
        root_y = find(y)
        
        if root_x == root_y:
            return
        
        if rank[root_x] > rank[root_y]:
            parent[root_y] = root_x
        elif rank[root_x] < rank[root_y]:
            parent[root_x] = root_y
        else:
            parent[root_y] = root_x
            rank[root_x] += 1
        
        min_edge_weight = min(min_edge_weight, distance_matrix[find(x), find(y)])
    
    for i in range(n):
        for j in range(i+1, n):
            if distance_matrix[i, j] < min_edge_weight:
                union(i, j)
    
    # Create the result matrix
    heuristic_matrix = np.full_like(distance_matrix, min_edge_weight)
    return heuristic_matrix
```

The cheapest_insertion() function first builds a minimum spanning tree using Kruskal's algorithm.
