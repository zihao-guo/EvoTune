```python
import numpy as np

def heuristics(distance_matrix):
    # Manhattan distance heuristic
    def manhattan_dist(node1, node2):
        return np.abs(node1[0] - node2[0]) + np.abs(node1[1] - node2[1])

    n_nodes = distance_matrix.shape[0]

    # Initialize the heuristic matrix with the same size as the input
    heuristic_matrix = np.full_like(distance_matrix, np.inf)

    # Calculate and populate the heuristic matrix
    for i in range(n_nodes):
        for j in range(n_nodes):
            if i!= j:
                # Manhattan distance heuristic
                heuristic_matrix[i, j] = distance_matrix[i, j] + manhattan_dist(i, j)

    # Normalize the heuristic matrix by dividing each element by the maximum heuristic value
    max_heuristic = np.max(heuristic_matrix)
    heuristic_matrix /= max_heuristic

    return heuristic_matrix
```

This function calculates a Manhattan distance heuristic for each edge in the graph and combines it with the input distance matrix. The normalized heuristic matrix provides a score for each edge, indicating how undesirable it is to include that edge in a solution.
