```python
def heuristics(distance_matrix):
    n = len(distance_matrix)
    # Calculate the triangle inequality heuristic
    heuristic_matrix = np.zeros((n, n))
    for i in range(n):
        for j in range(n):
            # For each edge (i,j), calculate how much the sum of distances to all other nodes (k) deviates from (i,j)
            deviation = sum(distance_matrix[i, k] for k in range(n) if k!= j) - distance_matrix[i, j]
            heuristic_matrix[i, j] = max(0, deviation)  # Non-negative value to avoid negative scores
    return heuristic_matrix
```

This new function considers the overall structure of the graph by comparing the sum of distances to other nodes for every edge. If this sum is significantly higher than the actual edge distance, we assign a higher heuristic score. This approach takes into account global properties of the distance matrix, which might not be fully captured by the simple distance itself.
