```python
def heuristics(distance_matrix):
    n = len(distance_matrix)
    heuristics_matrix = [[0.0] * n for _ in range(n)]
    
    # Calculate the sum of distances for each edge
    for i in range(n):
        for j in range(i+1, n):
            heuristics_matrix[i][j] = heuristics_matrix[j][i] = distance_matrix[i][j]
            heuristics_matrix[i][j] += distance_matrix[i][k] + distance_matrix[j][k] for k in range(n) if k!= i and k!= j
    
    # Normalize the heuristics matrix to values between 0 and 1
    min_value = min(min(row) for row in heuristics_matrix)
    max_value = max(max(row) for row in heuristics_matrix)
    heuristics_matrix = [[(x - min_value) / (max_value - min_value) for x in row] for row in heuristics_matrix]
    
    return heuristics_matrix
```

This function first computes the sum of distances for each edge, then normalizes these values to be within the range [0, 1], which should improve the scores compared to the previous methods. The idea is that edges connecting nodes with a higher total 'neighbor interaction' should be considered less desirable. The normalization step ensures the values are on the same scale, aiding better comparison.
