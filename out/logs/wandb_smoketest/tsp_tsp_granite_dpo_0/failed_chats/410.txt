```python
import numpy as np
from scipy.sparse.csgraph import csgraph_betweenness_centrality

def heuristics(distance_matrix):
    # Convert the distance matrix into a sparse, undirected graph
    n = distance_matrix.shape[0]
    graph = csgraph_betweenness_centrality(csgraph=distance_matrix, directed=False)
    
    # Create an empty matrix for the heuristics
    heuristics_matrix = np.zeros_like(distance_matrix)
    
    # Calculate edge importance scores
    for i in range(n):
        for j in range(i+1, n):
            # If the edge exists (distance[i][j] > 0), use betweenness centrality
            if distance_matrix[i,j] > 0:
                heuristics_matrix[i,j] = graph[i,j]
                heuristics_matrix[j,i] = graph[i,j]  # For symmetry
    
    # Normalize the heuristics matrix by total betweenness centrality
    max_betweenness = max(np.max(graph))
    heuristics_matrix = heuristics_matrix / max_betweenness
    
    return heuristics_matrix
```

This function calculates the betweenness centrality for each edge in the graph, where the weight of an edge is the distance from one vertex to another. This betweenness centrality score indicates how often an edge lies on the shortest path between all pairs of vertices. The higher the centrality, the more critical the edge is to the connectivity of the network, and hence, it should be treated as less desirable.
