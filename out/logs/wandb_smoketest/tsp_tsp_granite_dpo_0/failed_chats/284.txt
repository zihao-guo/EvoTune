```python
def heuristics(distance_matrix, edge_weights):
    num_nodes = distance_matrix.shape[0]
    heuristic_matrix = np.zeros_like(distance_matrix)
    
    # D^2 heuristic
    squared_distances = np.square(distance_matrix)
    d_squared = squared_distances.sum(axis=1)
    
    # Weighted D^2 for each edge
    for i in range(num_nodes):
        for j in range(i + 1, num_nodes):
            heuristic_matrix[i, j] = heuristic_matrix[j, i] = d_squared[i] + d_squared[j] * edge_weights[i][j]
    
    return heuristic_matrix
```

In this function, `edge_weights` is a 2D array that provides the importance of each edge, where `edge_weights[i][j]` indicates the weight associated with the edge between nodes `i` and `j`.
