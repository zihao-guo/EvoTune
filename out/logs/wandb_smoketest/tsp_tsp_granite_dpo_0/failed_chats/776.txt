In a weighted graph, the cost-to-come from a node 'i' to any other node 'j' is simply the distance 'd[i][j]' in our matrix. This metric captures the idea of how "bad" it is to include an edge connecting node 'i' and node 'j' because it signifies the highest potential total cost through that edge.

```python
def heuristics(distance_matrix):
    num_nodes = len(distance_matrix)
    # Initialize heuristic matrix
    heuristics_matrix = [[0 for _ in range(num_nodes)] for _ in range(num_nodes)]

    # Calculate cost-to-come for each node
    for i in range(num_nodes):
        for j in range(num_nodes):
            if i!= j:  # Ignore self-loops
                heuristics_matrix[i][j] = distance_matrix[i][j]  # Distance as is

    return heuristics_matrix
```
