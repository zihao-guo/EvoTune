```python
import numpy as np

def heuristics(distance_matrix):
    # To account for potential different weights (e.g., cost, time, etc.)
    weights = np.ones(distance_matrix.shape)  # Using uniform weights for simplicity

    # Calculate total weighted distance for each edge
    total_weighted_distances = distance_matrix + weights  # Adding weights to each element of the distance matrix

    # Sum these values for each row (edge), creating a new matrix
    row_sums = np.sum(total_weighted_distances, axis=1)

    # Normalize these values so that they sum to 1 for each row
    normalized_row_sums = row_sums / np.sum(row_sums)

    return normalized_row_sums
```

We then calculated the total weighted distance for each edge by adding the weight to the original distance.
