```python
def heuristics(distance_matrix):
    num_nodes = distance_matrix.shape[0]
    heuristics_matrix = np.zeros_like(distance_matrix)

    # Update heuristics for each node
    for i in range(num_nodes):
        for j in range(num_nodes):
            # Avoid diagonal elements
            if i == j:
                heuristics_matrix[i, j] = 0
            else:
                heuristics_matrix[i, j] = sum(distance_matrix[i, k] + distance_matrix[k, j] for k in range(num_nodes))

    # Normalize to have values between 0 and 1
    max_distance = np.max(heuristics_matrix)
    heuristics_matrix = heuristics_matrix / max_distance

    return heuristics_matrix
```

This function calculates a grid-aware 'Manhattan' distance between all pairs of nodes, then normalizes these values to fall within the 0 to 1 range. The higher the heuristic value for a particular edge, the more undesirable it is to include that edge in the solution.
