```python
def heuristics(distance_matrix, travel_frequency, edge_weights):
    n = len(distance_matrix)
    # Initialize the heuristic matrix with the same shape as the input
    heuristic_matrix = [[0 for _ in range(n)] for _ in range(n)]
    
    # Compute the heuristic score for each edge
    for i in range(n):
        for j in range(n):
            # Distance component
            heuristic_matrix[i][j] += distance_matrix[i][j]
            
            # Frequency component
            heuristic_matrix[i][j] -= travel_frequency[i][j]
            
            # Weight component
            heuristic_matrix[i][j] += edge_weights[i][j]
            
            # Alternative edge component
            for k in range(n):
                if k!= j:
                    heuristic_matrix[i][j] -= distance_matrix[i][k]
            
            # Normalize the heuristic score for better comparison
            total_heuristic = sum(sum(row) for row in heuristic_matrix)
            heuristic_matrix[i][j] /= total_heuristic
    
    return heuristic_matrix
```

In this function:

- `distance_matrix` is the input, a 2D list representing the distances between nodes.
- `travel_frequency` is a 2D list, where `travel_frequency[i][j]` denotes how often edge (i,j) is traveled.
- `edge_weights` is a 2D list representing the weights of the edges.

The function returns a heuristic matrix where the value at `heuristic_matrix[i][j]` signifies the undesirability of including edge (i,j) in a solution.
