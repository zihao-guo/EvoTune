Here's a Python function using NetworkX library to calculate edge weights based on betweenness centrality:

```python
import numpy as np
import networkx as nx

def heuristics(distance_matrix):
    G = nx.from_numpy_array(distance_matrix)
    # Calculate betweenness centrality
    betweenness_centrality = nx.betweenness_centrality(G)

    # Convert centrality values to heuristic scores
    heuristic_scores = np.zeros(distance_matrix.shape)
    for i in range(distance_matrix.shape[0]):
        for j in range(distance_matrix.shape[1]):
            heuristic_scores[i,j] = distance_matrix[i,j] + betweenness_centrality[i,j]

    return heuristic_scores
```
