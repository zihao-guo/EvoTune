```python
def heuristics(distance_matrix):
    # Compute the inverse of the diagonal elements (distance to itself)
    diag_inv_distance = 1 / distance_matrix.diagonal()

    # Compute the sum of each row (distance to all other nodes)
    row_sums = distance_matrix.sum(axis=1)

    # Apply the inverse distance to each element in the matrix
    heuristic_matrix = diag_inv_distance - row_sums

    return heuristic_matrix
```

In this function:

1. We first compute the inverse of the diagonal elements (distance to itself), giving more importance to nodes that are close to each other.
2. Next, we calculate the sum of each row, giving us an idea of the "closeness" to other nodes.
3. The heuristics are then calculated as the difference between the inverse diagonal element and the sum of the corresponding row. This results in a matrix where lower values indicate more desirable edges, based on the "easier to traverse" heuristic.
