```python
def heuristics(distance_matrix):
    # Calculate the average distance of each node's neighbors
    average_neighbor_distances = np.mean(distance_matrix[np.nonzero(distance_matrix)], axis=1)
    
    # Initialize the heuristics matrix with the same shape as the distance matrix
    heuristics_matrix = np.copy(distance_matrix)
    
    # Assign negative values to edges with high weight and high average distance
    for i in range(len(distance_matrix)):
        for j in range(len(distance_matrix)):
            if i!= j:  # don't consider diagonal elements
                # Avoid division by zero
                if distance_matrix[i, j] > 0:
                    heuristics_matrix[i, j] = -(distance_matrix[i, j] * average_neighbor_distances[j])
    
    return heuristics_matrix
```

1. Higher edge weights have a negative impact on their heuristic score.
