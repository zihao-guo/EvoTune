```python
def heuristics(distance_matrix):
    n = len(distance_matrix)
    heuristic_matrix = np.zeros((n, n))

    for i in range(n):
        for j in range(i+1, n):
            for k in range(n):
                if distance_matrix[i][k] + distance_matrix[k][j] < distance_matrix[i][j]:
                    # If direct path from i to j is not the shortest, 
                    # consider alternative paths through j
                    heuristic_matrix[i][j] += (distance_matrix[i][k] + distance_matrix[k][j])
    
    # To avoid negative scores, apply a minimum value
    np.maximum(heuristic_matrix, 0, out=heuristic_matrix)
    
    return heuristic_matrix
```

This heuristic considers the possibility of an alternative path (through the intermediate node j) that could reduce the total path cost. If such a path exists, this function adds its cost to the edge (i, j). The minimum value ensures that the scores won't go negative, maintaining numerical stability.
