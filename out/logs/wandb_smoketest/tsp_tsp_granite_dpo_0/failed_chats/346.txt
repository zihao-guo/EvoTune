The clustering coefficient helps identify highly interconnected parts of the graph, which may not be desirable. A high clustering coefficient indicates a high likelihood of triangles, making the edges between these nodes less desirable.

The diameter provides an indication of the "farthest" any two nodes are from each other. If this value is high, it may suggest that removing edges will significantly reduce the overall distance, which could be beneficial for our heuristic.

The entropy of the edge weights is a measure of their randomness or unpredictability. If the edge weights are highly predictable, it might be advantageous to remove edges with less unpredictability.

```python
import numpy as np

def heuristics(distance_matrix):
    n = distance_matrix.shape[0]
    
    # 1. Clustering Coefficient
    clustering_coef = np.diag(np.sum(distance_matrix**2, axis=1)) - np.sum(distance_matrix**2, axis=1)
    clustering_coef = clustering_coef / (n * np.sum(distance_matrix**2) - np.sum(distance_matrix**2, axis=1)**2)
    clustering_coef = np.maximum(clustering_coef, 0)  # Ensure non-negative values
    
    # 2. Diameter
    diameter = np.max(np.min(distance_matrix, axis=1))
    
    # 3. Entropy of Edge Weights
    edge_weights = distance_matrix
    entropy_weight = -np.sum(edge_weights * np.log(edge_weights))
    
    return np.array([
        clustering_coef,
        diameter,
        entropy_weight / np.log(n)  # Normalize entropy by log(n) for better comparison across different sizes of graphs
    ])
```
