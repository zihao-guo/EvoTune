```python
def heuristics(distance_matrix):
    num_nodes = distance_matrix.shape[0]
    
    # Calculate the cost-to-come for each edge
    cost_to_come = np.inf * np.ones((num_nodes, num_nodes))
    for u in range(num_nodes):
        for v in range(num_nodes):
            if u == v:
                cost_to_come[u, v] = 0  # No cost to come back to the same node
            else:
                for w in range(num_nodes):
                    cost_to_come[u, v] = min(cost_to_come[u, v], distance_matrix[u, w] + distance_matrix[w, v])
    
    # Normalize cost-to-come to get the heuristic values
    for u in range(num_nodes):
        for v in range(num_nodes):
            if u!= v:
                cost_to_come[u, v] /= (num_nodes - 1)  # Normalize by number of nodes - 1
    
    return cost_to_come
```

This function follows these steps:

1. Initialize `cost_to_come` with a very high value (infinity) to ensure any realistic edge distances will be lower.
2. For each node pair (u, v), calculate the minimum sum of distances from u to v and through any intermediate node w.
3. Normalize the resulting values for each edge (u, v) by dividing by `num_nodes - 1` to achieve a maximum value of 1.
