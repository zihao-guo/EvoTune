This measure quantifies the number of shortest paths that pass through a given edge. The higher the betweenness centrality, the more crucial the edge is for the overall connectivity of the graph.

```python
import numpy as np
import networkx as nx

def heuristics(distance_matrix):
    G = nx.from_numpy_array(distance_matrix)
    betweenness = nx.edge_betweenness_centrality(G, weight='weight') # Assuming 'weight' column in distance_matrix

    # Convert dictionary to numpy array for consistent dimensions
    betweenness_matrix = np.array(list(betweenness.values())).reshape(distance_matrix.shape)

    return betweenness_matrix
```
