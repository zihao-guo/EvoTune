To create a more sophisticated heuristics function, we'll incorporate a local optimization approach known as the Lin–Kernighan–Helsgaun (LKH) algorithm. This algorithm is commonly used for solving the Traveling Salesman Problem (TSP), but it can be adapted for other graph optimization problems as well. The main idea is to iteratively improve a partial solution by repeatedly swapping edges to reduce the total distance.

Here's a high-level overview of how we can implement this:

1. Initialize a random partial solution by selecting a random subset of edges.
2. Iterate over the remaining edges not in the partial solution.
3. For each edge, check if swapping it with the edge directly connected to the opposite endpoint (i.e., the edge we just didn't swap in step 2) improves the total distance.
4. If the swap improves the solution, perform the swap.

Now, let's implement this in Python. Note that this is a simplified version of the LKH algorithm and does not include error handling, edge case management, or extensive optimization.

```python
import numpy as np

def LKH_heuristics(distance_matrix):
    n = distance_matrix.shape[0]
    solution = np.random.permutation(n)  # Random partial solution
    prev_dist = np.sum(distance_matrix[solution[0], solution[1:]])  # Initial distance

    while True:
        improved = False
        for i in range(n):
            for j in range(i+1, n):
                if solution[i]!= solution[j] and solution[i]!= (solution[j] + 1) % n and solution[j]!= (solution[i] + 1) % n:
                    curr_dist = prev_dist + distance_matrix[solution[i], solution[j]]
                    next_solution = np.insert(solution, j+1, i)
                    next_dist = np.sum(distance_matrix[next_solution[0], next_solution[1:]])
                    if curr_dist < next_dist:
                        solution = next_solution
                        prev_dist = curr_dist
                        improved = True
                        break
                if improved:
                    break
        
        if not improved:
            break
    
    # Create heuristic matrix
    heuristics_matrix = np.zeros((n, n))
    for i in range(n):
        for j in range(n):
            heuristics_matrix[i, j] = distance_matrix[solution[i], solution[j]] if i <= j else distance_matrix[solution[i], solution[(i+1) % n]]

    return heuristics_matrix
```
