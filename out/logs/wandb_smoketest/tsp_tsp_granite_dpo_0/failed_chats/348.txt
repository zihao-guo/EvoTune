```python
import numpy as np

def heuristics(distance_matrix):
    # Calculate row sums to represent the degree of each node
    row_sums = np.sum(distance_matrix, axis=1)
    
    # Calculate cumulative distribution of distances for each node
    cumulative_distance = np.cumsum(distance_matrix) / np.sum(distance_matrix, axis=1)[:, np.newaxis]
    
    # Calculate centrality for each edge
    heuristic_matrix = -np.square(cumulative_distance - np.roll(cumulative_distance, 1)) / (row_sums - 1)
    
    return heuristic_matrix
```

This function calculates two values for each edge:
1. Cumulative distance of the node from its two neighbors.
2. The difference between the cumulative distances of the two neighbors.

The 'heuristic' value for an edge is the negative of the squared difference, divided by the difference in the row sums of the neighbors. This normalization ensures that edges connecting nodes with similar neighborhoods will have higher values, indicating they are less desirable, while edges connecting nodes with diverse neighborhoods will have lower, more favorable values.
