```python
import numpy as np

def heuristics(distance_matrix):
    # Global properties: total edges and total distance
    n = distance_matrix.shape[0]
    total_edges = n*(n-1)//2  # Assuming no self-loops
    total_distance = distance_matrix.sum()

    # Local properties: average distance of each vertex
    local_averages = distance_matrix.mean(axis=1)

    # Heuristic scores
    heuristics_matrix = np.empty(distance_matrix.shape)

    for i in range(n):
        for j in range(i+1, n):
            heuristics_matrix[i, j] = heuristics_matrix[j, i] = distance_matrix[i, j]

            # Penalize if the current edge (i,j) or (j,i) is high
            heuristics_matrix[i, j] += local_averages[i] + local_averages[j]

            # Penalize if the edge is longer than the average edge length
            if distance_matrix[i, j] > (total_distance / total_edges):
                heuristics_matrix[i, j] += total_distance / total_edges
                heuristics_matrix[j, i] += total_distance / total_edges

    return heuristics_matrix
```

This function scores each edge based on two aspects:
1. The average distance of its endpoints (local information).
2. The global average distance (global information).
3. The ratio of the current edge's length to the average edge length in the graph (a measure of how "dense" the graph is).

The global aspect emphasizes edges that contribute to a high overall connectivity of the graph, while the local aspect penalizes edges to endpoints with high individual distances. The ratio term adds a bias against longer edges, favoring more balanced graphs.
