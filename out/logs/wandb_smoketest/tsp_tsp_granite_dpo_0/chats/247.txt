This method takes into account both the immediate distance (first-order deviation) and the deviation from the expected distance based on the weighted average (second-order deviation). The idea is to penalize deviations more heavily as they increase, making the heuristic more sophisticated than a simple re-weighting of the distances.

import numpy as np

def heuristics(distance_matrix):
    n = len(distance_matrix)
    # Calculate the weighted average distance
    weighted_avg = np.average(distance_matrix, axis=0)
    # Calculate the second-order deviation
    second_order_deviation = np.square(distance_matrix - weighted_avg)
    # Calculate the total weighted deviation
    total_deviation = np.sum(distance_matrix, axis=0) + np.sum(second_order_deviation)
    # Return the total weighted deviation as the heuristic values
    return total_deviation.reshape(-1, 1)
