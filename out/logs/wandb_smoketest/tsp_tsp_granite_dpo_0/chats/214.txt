```python
import numpy as np

def heuristics(distance_matrix):
    # Initialize the heuristic matrix with the same shape as the distance matrix
    num_nodes = distance_matrix.shape[0]
    heuristic_matrix = np.zeros((num_nodes, num_nodes))

    # Edge density calculation
    edge_density = np.sum(distance_matrix, axis=1) / (num_nodes - 1)
    heuristic_matrix += edge_density[:, np.newaxis] + edge_density[np.newaxis, :] - 2 * distance_matrix

    # Add edge weight normalization
    min_distance = np.min(distance_matrix)
    heuristic_matrix += (min_distance - distance_matrix) / min_distance

    # Normalization
    max_heuristic = np.max(heuristic_matrix)
    heuristic_matrix = (heuristic_matrix - np.min(heuristic_matrix)) / (max_heuristic - np.min(heuristic_matrix))

    return heuristic_matrix
```

This new function introduces two main steps:

1. Edge Density: It computes the average distance per node for each edge. This essentially captures how densely connected a set of nodes is. A densely connected set will naturally have higher edge weights, contributing more to the heuristic score.

2. Weight Normalization: It normalizes each edge's weight relative to the minimum distance, which will give more importance to nodes with significantly larger distances.

The final normalized matrix gives the heuristics for each edge, providing a more detailed insight into the undesirability of including an edge in a solution.
