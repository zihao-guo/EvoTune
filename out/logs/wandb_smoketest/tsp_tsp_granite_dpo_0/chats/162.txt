```python
import numpy as np

def heuristics(distance_matrix):
    # Compute the average of row sums for each row as a rough estimate of the diagonal length
    average_diagonal = np.mean(np.diagonal(distance_matrix))
    
    # Create a matrix of the same shape as the input distance matrix
    heuristic_matrix = np.zeros_like(distance_matrix)
    
    # Populate the heuristic matrix: a closer vertex will have a lower heuristic value
    for i in range(distance_matrix.shape[0]):
        for j in range(distance_matrix.shape[1]):
            heuristic_matrix[i][j] = distance_matrix[i][j] / average_diagonal  # Scale the distance by average diagonal
            heuristic_matrix[i][j] = 1 / (1 + heuristic_matrix[i][j])  # Apply a sigmoid-like function to ensure values are small

    return heuristic_matrix
```

This function works as follows:

1. It first calculates the average of the diagonal elements of the input distance matrix, which can be seen as a rough estimate of the ideal edge length (i.e., diagonal of a perfect "square").

2. Then, for each edge (i, j), it calculates a heuristic score by scaling the actual distance by this average diagonal and applying a sigmoid function. The sigmoid function ensures that the score remains small, particularly for longer distances.

3. The result is a heuristic matrix where closer edges have lower heuristic values, indicating they are more likely to be part of an optimal solution.
