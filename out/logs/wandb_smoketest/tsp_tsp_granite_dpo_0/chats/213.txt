```python
import numpy as np

def heuristics(distance_matrix):
    # Calculate the degree of each node (number of neighbors)
    degrees = np.sum(distance_matrix, axis=0)
    
    # Create a weight matrix where each entry represents the weight of the corresponding edge
    weight_matrix = np.zeros_like(distance_matrix)
    
    for i in range(distance_matrix.shape[0]):
        for j in range(distance_matrix.shape[1]):
            weight_matrix[i, j] = 1 / degrees[j]  # More connected nodes should have lower weights
    
    # Normalize weights to get a probability distribution
    weight_matrix = weight_matrix / np.sum(weight_matrix, axis=0)
    
    return weight_matrix
```

This function calculates the degree (number of neighbors) for each node and uses it as a weight to evaluate edge importance. Nodes with higher degrees get lower weights, making edges connecting them to less connected nodes more desirable. This strategy reflects the intuition that connecting to highly connected nodes could be beneficial in certain scenarios, such as network resilience or data transmission.
