To create a more effective heuristics() function, we'll use the concept of "Traveling Salesman Problem (TSP) heuristics" which aim to estimate the quality of a partial solution by giving lower values to edges that will likely be part of the optimal tour. The most popular TSP heuristics are:

1. Nearest Neighbor (NN)
2. 2-Opt
3. 3-Opt
4. Lin-Kernighan (LK)

For simplicity, I'll present a hybrid approach combining Nearest Neighbor and 2-Opt. This method assigns a heuristic score to each edge based on how well it fits into a smaller partial tour.

Here's the implementation of the `heuristics()` function:

```python
import numpy as np

def heuristics(distance_matrix):
    n = len(distance_matrix)
    # Step 1: Nearest Neighbor heuristic
    nn_heuristics = np.minimum.accumulate(distance_matrix, axis=1)
    
    # Step 2: 2-Opt heuristic (Finding the shortest cycle in the current partial tour)
    partial_tour = np.argsort(np.sum(distance_matrix, axis=1), axis=0)
    current_tour_distances = np.sum(distance_matrix[partial_tour])
    best_cycle_distance = current_tour_distances
    
    for i in range(n):
        for j in range(i + 1, n):
            if np.all(np.array([i, j, partial_tour[i], partial_tour[j]])!= np.array([i, j, partial_tour[j], partial_tour[i]])):  # Avoid repeated edges
                cycle_distances = np.sum(distance_matrix[partial_tour[i:j+1]])
                if cycle_distances < best_cycle_distance:
                    best_cycle_distance = cycle_distances
    
    # Step 3: Combine NN and 2-Opt heuristics
    heuristics_matrix = np.minimum(nn_heuristics, best_cycle_distance)
    return heuristics_matrix
```

This function first calculates the nearest neighbor heuristics and then uses 2-Opt to find the smallest possible cycle in the current partial tour. The final heuristic value for each edge is the minimum of the NN heuristic and the 2-Opt cycle distance.

By combining these two methods, we are encouraging the algorithm to consider both locally optimal solutions (NN heuristic) and globally optimal solutions (2-Opt), which should improve the performance over a simple accumulation of distances.
