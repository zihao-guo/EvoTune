Here's a Python function that implements this idea:

```python
def heuristics(distance_matrix):
    # Calculate the sum of each row (total capacity)
    total_capacity_row = np.sum(distance_matrix, axis=1)
    
    # Create a new matrix where each element indicates the improvement if this edge is included
    heuristic_matrix = total_capacity_row[:, np.newaxis] - distance_matrix
    
    # We want edges with higher heuristic values to be less desirable, so we take the absolute value
    heuristic_matrix = np.abs(heuristic_matrix)
    
    return heuristic_matrix
```

In this function:

1. `np.sum(distance_matrix, axis=1)` calculates the total capacity (distance) for each node, summing up the distances from every other node.

2. `np.newaxis` is used to add an extra dimension to the total capacity vector so that it can be broadcasted correctly with the original distance matrix.

3. The subtraction `total_capacity_row[:, np.newaxis] - distance_matrix` produces a matrix where each entry is the difference between the total capacity of the corresponding node and its distance to all other nodes. If this value is negative, the edge is desirable (i.e., we should include it), and if it's positive, the edge is undesirable.

4. `np.abs(heuristic_matrix)` converts these difference values into absolute values. The result is a heuristic matrix where higher values indicate better (less desirable) edges to be included in the solution.
