```python
import numpy as np

def heuristics(distance_matrix):
    # Calculate the average distance for each edge
    avg_distance = np.mean(distance_matrix, axis=0)
    
    # Calculate the sum of each row (edge density)
    row_sum = np.sum(distance_matrix, axis=1)
    
    # Create an empty matrix to store the heuristics
    heuristics_matrix = np.zeros_like(distance_matrix)
    
    # Assign heuristics based on the average distance and edge density
    heuristics_matrix = avg_distance - (distance_matrix / row_sum[:, np.newaxis])
    
    return heuristics_matrix
```

The function then subtracts the proportion of a weight from the average weight. This subtraction is used to indicate the undesirability of including an edge.
