```python
import numpy as np

def heuristics(distance_matrix):
    # Create a mask for edges
    n = distance_matrix.shape[0]
    edge_mask = np.ones((n, n), dtype=bool)
    np.fill_diagonal(edge_mask, False)
    
    # Calculate distance-based and proximity-based scores
    distance_scores = distance_matrix
    proximity_scores = np.minimum(distance_matrix, np.roll(distance_matrix, -1, axis=1)).min(axis=1)
    
    # Combine scores using a weighted sum
    weight = 0.8  # Adjust this parameter as needed
    heuristic_scores = weight * distance_scores + (1 - weight) * proximity_scores
    
    # Add randomness for diversity
    random_scores = np.random.rand(n, n)
    final_heuristic_scores = heuristic_scores + random_scores
    
    # Scale the final scores to ensure they sum to 1 (for easier interpretation)
    max_score = final_heuristic_scores.max()
    final_heuristic_scores = final_heuristic_scores / max_score
    
    return final_heuristic_scores
```

This function assigns both a distance-based and proximity-based score to each edge, then combines these with a weighted average and adds random scores to ensure a diverse, unbiased solution. The final scores are scaled to ensure they sum to 1, making them easier to interpret and compare with other heuristics.
