```python
import numpy as np

def heuristics(distance_matrix):
    # Calculate the inverse of distances
    inverse_distances = 1 / distance_matrix

    # Avoid negative or zero values (since we're dealing with distances)
    inverse_distances[inverse_distances < 0] = 0

    # Scale the inverse distances to sum up to 1
    total = inverse_distances.sum()
    if total!= 0:
        inverse_distances *= 1 / total

    return inverse_distances
```

This function first calculates the inverse of the input distance matrix. To avoid division by zero and ensure all values are non-negative, we replace any value less than zero with zero. Then, to ensure the row sums of the matrix equal one (which is a desirable property for a probability distribution), we scale the matrix by a factor that makes the row sums equal to one.
