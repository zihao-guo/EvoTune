```python
def heuristics(distance_matrix):
    # Calculate the distance between each pair of nodes
    num_nodes = len(distance_matrix)
    distances = numpy.triu(distance_matrix) # Consider only above diagonal elements for lower-triangular matrix
    row_ind, col_ind = numpy.where(distances > 0)

    # Define a risk factor as the absolute difference between distances divided by the maximum distance
    risk_factor = abs(distances[row_ind, col_ind]) / distance_matrix.max()

    # Create a heuristic matrix, where the diagonal elements are 0 (no cost) and higher off-diagonal elements represent riskier choices
    heuristic_matrix = numpy.zeros((num_nodes, num_nodes))
    heuristic_matrix[row_ind, col_ind] = risk_factor[row_ind] + risk_factor[col_ind]

    return heuristic_matrix
```

In this function:

- `distance_matrix` is a square numpy array where `distance_matrix[i, j]` is the distance from node `i` to node `j`.
- `numpy.triu()` is used to consider only the upper (or upper-triangular) triangle of `distance_matrix`, excluding the diagonal, as the cost to travel from node `i` to itself is 0.
- `numpy.where(distances > 0)` finds indices where the distances are non-zero.
- The `risk_factor` is the absolute difference between distances divided by the maximum distance, ensuring a uniform scale from 0 to 1.
- The resulting heuristic matrix values represent the risk (undesirability) of including an edge between each pair of nodes.
