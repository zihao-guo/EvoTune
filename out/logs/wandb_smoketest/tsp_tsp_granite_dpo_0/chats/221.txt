```python
import numpy as np

def heuristics(distance_matrix):
    # Get the shape of the input matrix
    n, _ = distance_matrix.shape
    
    # Initialize a similar shaped matrix to hold the heuristic values
    heuristic_matrix = np.zeros_like(distance_matrix)
    
    # Apply the Inverse Distance Heuristic
    for i in range(n):
        for j in range(n):
            if i!= j:
                # Calculate the inverse of the distance
                inv_distance = 1 / distance_matrix[i, j]
                # Assign the inverse distance as the heuristic value
                heuristic_matrix[i, j] = inv_distance
    
    return heuristic_matrix
```

This function iterates over each pair of nodes and calculates the inverse of their distance. This method prioritizes nodes that are farther apart as they are given higher heuristic values, which should lead to better performance in most cases.
