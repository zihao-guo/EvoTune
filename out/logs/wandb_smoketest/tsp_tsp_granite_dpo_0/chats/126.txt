```python
import numpy as np

def heuristics(distance_matrix):
    # Convert distance matrix to adjacency matrix
    adj_matrix = np.where(distance_matrix < np.inf, 1, 0)
    
    # Calculate adjacency matrix's eigenvector centrality
    eig_centrality = np.linalg.eigvals(adj_matrix.T @ adj_matrix)
    eig_centrality = np.real(eig_centrality) # Only consider real parts
    eig_centrality /= np.linalg.norm(eig_centrality)  # Normalize

    # Calculate distance-weighted average for each node
    node_importance = np.average(distance_matrix, axis=1, weights=np.average(distance_matrix, axis=0))

    # Combine eigenvector centrality and distance-weighted average
    heuristic_matrix = np.column_stack((eig_centrality, node_importance))

    return heuristic_matrix
```

This function first converts the input distance matrix into an adjacency matrix, where edges are represented as 1s and non-existent edges as 0s. Then it calculates the eigenvector centrality of the adjacency matrix. The eigenvector centrality gives us the influence or importance of each node (edge) in the network.

Next, it computes a distance-weighted average for each node. This helps us understand the average 'importance' of all connected edges for each node.

Finally, it combines these two indicators to create a heuristic matrix, where each row indicates both the eigenvector centrality of a node (edge) and its distance-weighted average importance. This approach should yield a more nuanced understanding of each edge's desirability, potentially leading to a higher score.
