This principle suggests that the path through an intermediate vertex is likely to be shorter than going directly from one vertex to another. 

Let's first understand how the triangle inequality is related to the distance matrix:

- For each edge (i, j) in the graph, if i < j, the value at position (i, j) in the distance matrix (d_ij) should be greater than d_ii + d_jj.
- For edges where i > j, d_ij should be less than or equal to d_ii + d_jj.

def heuristics(distance_matrix):
    num_vertices = distance_matrix.shape[0]

    # Initialize a new matrix to store the heuristic scores
    heuristics_matrix = distance_matrix.copy()

    # Apply the triangle inequality principle
    for i in range(num_vertices):

            if distance_matrix[i, j] < distance_matrix[i, i] + distance_matrix[j, j]:
                heuristics_matrix[i, j] = distance_matrix[i, i] + distance_matrix[j, j]
            elif distance_matrix[i, j] > distance_matrix[i, i] + distance_matrix[j, j]:
                heuristics_matrix[i, j] = distance_matrix[i, j]

    return heuristics_matrix
