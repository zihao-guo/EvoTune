This method assigns the lowest distance among all remaining unvisited vertices to each edge. def heuristics(distance_matrix):
```python
    n = len(distance_matrix)  # assuming the matrix is square
    heuristic_matrix = distance_matrix.copy()  # copy the input matrix to avoid modifying it

    for i in range(n):
        for j in range(i + 1, n):  # iterate over unvisited vertices
            heuristic_matrix[i][j] = distance_matrix[i][j]
            heuristic_matrix[j][i] = distance_matrix[j][i]  # for symmetry

    return heuristic_matrix
```

This function assigns the smallest distance for each pair of vertices to its corresponding edge, as long as it hasn't been set yet. The symmetry is maintained by assigning the same value to each edge.
