```python
import numpy as np

def heuristics(distance_matrix):
    # Determine the number of cities
    num_cities = distance_matrix.shape[0]

    # Initialize the heuristic matrix with the distance matrix
    heuristic_matrix = distance_matrix.copy()

    # Create a list of indices for unvisited cities
    unvisited_indices = np.arange(num_cities)

    # Iterate through each city (starting from the first one)
    for i in range(num_cities):
        # Calculate the nearest unvisited city's heuristic score
        nearest_unvisited_idx = unvisited_indices[np.argmin(heuristic_matrix[i, unvisited_indices])]
        nearest_distance = distance_matrix[i, nearest_unvisited_idx]

        # Update the heuristic score for the current city
        heuristic_matrix[i, i] = nearest_distance

        # Remove the visited city from the list of unvisited indices
        unvisited_indices = unvisited_indices[unvisited_indices!= nearest_unvisited_idx]

    return heuristic_matrix
```

This `heuristics` function calculates a novel score for each edge, which indicates the shortest hypothetical path between two cities (considering the current unvisited cities). The function assumes that the optimal tour would pass through the nearest unvisited city. This heuristic is often more efficient than the previous functions as it significantly reduces the number of edges to consider due to the sequential optimization from the first city to the last.
