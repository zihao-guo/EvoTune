```python
import numpy as np

def heuristics(distance_matrix):
    # Calculate the inverse of distance matrix
    inv_dist_matrix = 1 / distance_matrix
    
    # Normalize the inverse matrix to sum up to 1 along each row
    row_sums = np.sum(inv_dist_matrix, axis=1)
    normalized_inv_dist_matrix = inv_dist_matrix / row_sums[:, np.newaxis]
    
    # Return the normalized inverse distance matrix as heuristic scores
    return normalized_inv_dist_matrix
```

This function normalizes each row of the inverse distance matrix so that the sum equals 1. This normalization allows for a comparison between different-sized matrices and provides a score for each edge that reflects its inverse weighted importance.

This heuristic takes into account the weight (reversing the distance matrix) and connectivity of the graph (normalizing by row sums). These features aim to balance the selection of edges, favoring denser, shorter-weighted connections, which are often more significant in network structures.

The performance improvement of this function compared to the initial one heavily depends on the specific characteristics of the input distance matrix. Higher-weighted edges, denser structures, and more balanced weight distributions will yield better performance according to this heuristic.
