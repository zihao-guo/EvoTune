```python
def heuristics(distance_matrix):
    num_nodes = distance_matrix.shape[0]
    impact_matrix = np.full((num_nodes, num_nodes), 0.0)
    
    # Add positive impact for edges that connect high-impact nodes
    for i in range(num_nodes):
        for j in range(i+1, num_nodes):
            distance = distance_matrix[i, j]
            impact_i = impact_matrix[i, :]  # sum of impacts from i to all nodes
            impact_j = impact_matrix[j, :]  # sum of impacts from j to all nodes
            
            # If i and j are high-impact nodes, skip other connections to i and j to increase the benefit of connection
            if np.sum(impact_i) > 10 or np.sum(impact_j) > 10:
                impact_i[i] += 1
                impact_j[j] += 1
            
            impact_matrix[i, j] = distance
            impact_matrix[j, i] = distance
    
    # Normalize impact_matrix
    impact_matrix = impact_matrix / np.max(impact_matrix)
    
    return impact_matrix
```

This function introduces two aspects for each edge:
1. Direct distance (from the original matrix).
2. The potential for benefit, determined by the sum of impacts from the two nodes involved.

This approach might outperform the simple matrix-copy function because it encourages connections that link high-impact nodes, prioritizing those connections over direct distance alone.

The parameters (10 in this example) can be adjusted based on how you define "high-impact" nodes. A high number indicates that the function values connections from nodes that have significant influence more, and the low-impact nodes' edges are given less weight.
