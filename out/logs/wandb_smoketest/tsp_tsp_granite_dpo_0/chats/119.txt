To achieve a better heuristics function, we'll use a common approach in mathematical optimization known as the Traveling Salesman Problem (TSP) heuristics: the nearest neighbor (NN) or nearest insertion (NI) algorithm. This method selects the nearest unvisited node at each step, which often leads to good solutions, especially when the input is a complete graph.

Below is a Python function implementing the nearest neighbor heuristics:

```python
import numpy as np

def heuristics(distance_matrix):
    # Ensure the input is a numpy array for easier operations
    distance_matrix = np.array(distance_matrix)
    
    n = distance_matrix.shape[0]
    # Initialize the result matrix with zeros
    heuristic_scores = np.zeros_like(distance_matrix)
    
    # Iterate over each node
    for i in range(n):
        # Find the closest unvisited node
        closest_index = np.argmin(distance_matrix[i, np.nonzero(distance_matrix[i, :])[0]])
        
        # Assign the distance to the closest node as the heuristic score for the current node
        heuristic_scores[i, closest_index] = distance_matrix[i, closest_index]
    
    return heuristic_scores
```

This function calculates the nearest neighbor heuristics for the TSP by visiting each node in turn and selecting the closest unvisited node as the next step in the tour. This leads to reasonably good solutions, especially for complete graphs.
