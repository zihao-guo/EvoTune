```python
def heuristics(distance_matrix):
    # Get matrix dimensions
    num_nodes = distance_matrix.shape[0]
    
    # Initialize heuristic matrix with zeros
    heuristic_matrix = np.zeros((num_nodes, num_nodes))
    
    # Calculate heuristic score for each edge
    for i in range(num_nodes):
        for j in range(i, num_nodes):  # Only consider upper triangle (i < j) to avoid double counting
            # Weight of the edge (i, j)
            edge_weight = distance_matrix[i, j]
            
            # Adjacent edges of edge (i, j)
            adj_i = np.where(distance_matrix[i, :] < np.inf)[0]  # Find adjacent nodes for i
            adj_j = np.where(distance_matrix[:, j] < np.inf)[0]  # Find adjacent nodes for j
            
            # Sum of weights of adjacent edges of (i, j)
            adj_sum = np.sum(distance_matrix[adj_i, adj_j])
            
            # Calculate heuristic score
            heuristic_matrix[i, j] = heuristic_matrix[j, i] = edge_weight + adj_sum
    
    return heuristic_matrix
```

This function calculates a heuristic score for each edge by considering both its weight and the total weight of its adjacent edges. This should lead to better performance because it takes into account the network's interconnectedness, in addition to individual edge costs.
