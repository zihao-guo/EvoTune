```python
import numpy as np

def heuristics(distance_matrix):
    # Get the number of vertices
    num_vertices = distance_matrix.shape[0]
    
    # Calculate the mean and standard deviation of distances
    mean_distance = np.mean(distance_matrix)
    std_distance = np.std(distance_matrix)
    
    # Initialize the heuristic matrix with zeros
    heuristic_matrix = np.zeros(distance_matrix.shape)
    
    # Calculate heuristic for each edge
    for i in range(num_vertices):
        for j in range(i+1, num_vertices):
            # Edge weight (distance)
            edge_weight = distance_matrix[i, j]
            
            # Calculate heuristic as a function of mean and standard deviation
            # and the edge weight
            heuristic_matrix[i, j] = heuristic_matrix[j, i] = (edge_weight - mean_distance) / std_distance
    
    # Normalize heuristic values
    heuristic_matrix = (heuristic_matrix - np.min(heuristic_matrix)) / (np.max(heuristic_matrix) - np.min(heuristic_matrix))
    
    return heuristic_matrix
```

In this revised function, we calculate the mean and standard deviation of the distances in the matrix. The heuristic for each edge is then determined by how far the edge weight is from the mean, normalized by the standard deviation. This captures the potential variability or undesirability of the edge compared to typical distances in the graph.

This change leverages statistical analysis, which should make the function perform better than the simple reimplementation of the distance matrix.
