```python
def heuristics(distance_matrix):
    num_nodes = distance_matrix.shape[0]
    heuristic_matrix = np.zeros(distance_matrix.shape)
    
    for i in range(num_nodes):
        for j in range(i+1, num_nodes):
            distance = distance_matrix[i, j]
            heuristic_matrix[i, j] = heuristic_matrix[j, i] = 1 / distance  # Assuming the same distance in both directions
    
    return heuristic_matrix
```

This function will return a matrix where the value at each position (i, j) represents the inverse of the distance between nodes i and j. Thus, it gives a higher heuristic score to shorter distances, which implies that it considers those edges to be less undesirable.
